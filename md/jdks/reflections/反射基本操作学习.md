### 反射API

#### 1. 如何获取Class对象？

- 方法一
```
   Class class = Foo.class; 
```

- 方法二
```
Foo foo1 = new Foo();
Class class = foo1.getClass();
```

- 方法三

```
  class.forName("com.someecho.reflections.domain.Foo")
```

#### 2. Class类型有哪些操作？

- 获取包名
- 获取class名
- 获取方法
- 获取接口
- 获取构造函数
- 获取类加载器
- 获取成员变量
- 获取注解
